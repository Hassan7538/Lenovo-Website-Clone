Windows Update

  1. Close all applications & restart your computer to stop background applications.
  2. Click on the Start menu and select the Settings icon.
  3. Click on Update & Security, then on check for updates.
  4. Or if you see Check online for updates from Microsoft Update, click on it.
  5. Available updates will download automatically, but if you see a Download or Install button, click on it.

Windows Installation

  1. Open any browser and type microsoft window download.
  2. From there click on software download and select your desired window.
  3. Click Download tool now to download the file MediaCreationTool.exe
  4. Accept the license terms
  5. Select Create installation media (USB flash drive, DVD, or ISO file) for another PC, and then click Next.
  6. Now your bootable drive is created. Insert it into another PC.
  7. Choose your language, timezone, and keyboard settings etc & then click on next.
  8. Click Install Now and select the windows. Click Next to start the installation process
  9. Now select the partition where you want to install Windows and click Next. The Windows 10 installation will begin. Your computer may reboot a few times during the process.
  10. After installation, your computer will automatically restart one last time. When it starts up again for the first time, you'll be able to choose more of your settings or use express settings recommended by Microsoft.
  11. Finally, sign in your to your microsoft account, or you may skip this step.
  12. Now you are ready to go...
